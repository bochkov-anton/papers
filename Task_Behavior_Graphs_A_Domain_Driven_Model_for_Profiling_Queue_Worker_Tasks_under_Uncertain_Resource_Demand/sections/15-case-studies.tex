\section{Case Studies}
\label{sec:case-studies}

This section outlines worked examples used to validate the model structure. A
replication case study should show how payload size, peer health, network
condition, and retry state influence network output, connection-hold time, and
worker-slot time. A compute-transform case study should contrast high active
CPU demand with lower occupancy sensitivity. An external-dependency case study
should show low CPU demand but high occupancy under degraded dependencies.

The current paper treats these as structured examples rather than empirical
production results.
