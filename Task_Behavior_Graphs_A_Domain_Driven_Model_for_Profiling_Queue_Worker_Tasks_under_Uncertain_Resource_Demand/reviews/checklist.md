# Review Checklist

- [ ] The paper states the problem clearly.
- [ ] The contribution is not merely a metric taxonomy.
- [ ] Task Behavior Graphs are defined precisely.
- [ ] Worker occupancy is treated as a first-class cost.
- [ ] Entropy, confidence, and consequence are separated.
- [ ] The manifest model is concrete enough to implement.
- [ ] The paper avoids claiming global metric completeness.
- [ ] Related work is positioned fairly.
